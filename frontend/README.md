# Frontend - NOTESIA

Esta carpeta contendrá el frontend de la aplicación NOTESIA desarrollado en React.

## Estructura del Proyecto

```
frontend/
├── src/
│   ├── components/
│   ├── pages/
│   ├── hooks/
│   ├── services/
│   ├── utils/
│   └── App.jsx
├── public/
├── package.json
└── vite.config.js
```

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- React Router
- Axios

## Configuración

El frontend se desplegará por separado del backend en Vercel.
